# 🏕️ Участки

Участки могут пренадлежать жителю города, защищая его имущество от других жителей города. Также у каждого участка есть собственные права, которые можно регулировать для определённых групп игроков.

## <mark style="color:$primary;">Команды</mark>

* `/plot claim` - купить участок
* `/plot give <ник>` - выдать участок определённому игроку
* `/plot forsale <цена>` - выставить участок на продажу
* `/plot notforesale/nfs` - снять участок с продажи
* `/plot unclaim` - перестать быть владельцем своего участка
* `/plot set price <кол-во>` - установить цену участка
* `/plot set name <name>` - установить имя участка
* `/plot perms` - меню управления правами участка, в котором находишься
* `/plot set perm reset` - выставить все права для всех групп по умолчанию
* `/plot set perm <вкл/выкл>` - изменить все права, для всех группа
* `/plot set perm <группа> <вкл/выкл>` - изменить все права, для определённой группы
* `/plot set perm <право> <вкл/выкл>` - изменить определённое право для всех групп
* `/plot set perm <группа> <право> <вкл/выкл>` - изменить определённое право, для определённой группы
* `/plot evict` - выселить игрока из участка
* `/plot flags` - меню управления разрешениями участка, в котором находишься
* `/plot toggle <разрешение>` - изменить разрешение участка на противоположное значение
* `/plot toggle <разрешение> <on/off>` - изменить разрешение участка на определённое значение
* `/plot group <такие подкоманды как у участка>` - команды для группы участков

## <mark style="color:$primary;">Выдача участка жителю</mark>

Жители города должны где-то жить, и при этом иметь защиту своей территории и от грабежа со стороны других жителей. Для этой цели существуют участки.

Передать участок другому жителю можно двумя способами: Передать бесплатно и Продать:

1. Чтобы передать бесплатно участок жителю используй команду:

* **`/plot give <ник>`** - выдать определённому жителю участок, внутри которого ты находишься

2. Чтобы продать участок другому жителю нужно выставить участок на продажу

* **`/plot set forsale <цена>`** - выставить участок на продажу за определённую цену, можно указать 0, тогда продажа будет бесплатной
* житель города должен прописать команду `/plot claim` внутри этого участка, чтобы его купить

Теперь у твоего жителя собственная земля. Другие жители города не смогут ломать/строить/смотреть\_сундуки на его территории.

## <mark style="color:$primary;">Добавление игрока в приват</mark>

Если ты хочешь дать доступ к своему участку другим игрокам, то тебе нужно добавить их в друзья:

* **`/t friend add <ник>`** - добавить игрока в друзья
* **`/t friend remove <ник>`** - удалить игрока из друзей
* **`/t friends`** - список своих друзей

{% hint style="warning" %}
Максимальное кол-во друзей равно **10**
{% endhint %}

## <mark style="color:$primary;">Группы участков</mark>

Участки можно объденить в группы и настраивать их без необходимости бегать по каждому участку.

Создать группу можно командой:

* **`/plot group create <название>`** - создать группу
* **`/plot group delete <группа>`** - удалить группу(участки не удаляются)

Чтобы добавить участок в группу, нужно встать в этот участок и выполнить команду:

* **`/plot group add <группа>`** - добавить участок в группу
* `/plot group remove`- удалить участок, на котором стоишь, из группы&#x20;

Управление участками в группе происходит аналогичными комадами, как и просто одним участком, только к команде нужно добавлять слово `group` :

* **`/plot group forsale <группа> 12`** - выставить все участки в группе на продажу за 12 монет
* **`/plot group give <группа> <ник>`** - сделать игрока владельцем во всех участках в группе
* **`/plot group perms <группа>`** - настройка разрешений для всех участков группы
* **`/plot group flags <группа>`** - настройка флагов для всех участков в группе
* и так далее
