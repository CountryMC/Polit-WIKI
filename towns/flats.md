# 🚪 Квартиры

## <mark style="color:$primary;">Команды</mark>

* `/t flats` - список квартир в городе
* **`/flat`** - меню управления квартирой в которой стоишь
* **`/flat create <название>`** - создать квартиру с определённым названием
* **`/flat delete`** - удалить квартиру
* `/flat forsale` - выставить/снять квартиру на продажу
* **`/flat set price <цена>`** - установить цену квартиры
* **`/flat set name <цена>`** - изменить название квартиры
* `/flat member add <ник>` - добавить жильца в квартиру
* `/flat member remove <ник>` - удалить жильца из квартиры
* `/flat owner set <ник>` - сделать игрока владельцем квартиры
* `/flat owner clear` - удалить владельца из квартиры, но не выселить
* **`/flat buy`** - купить квартиру
* **`/flat show`** - показать границы квартиры
* **`/flat move`** - передвинуть голограму в то место, в котором находишься

## <mark style="color:$primary;">Создание квартиры</mark>

Для создания квартиры нужно выполнить несколько простых шагов:

{% stepper %}
{% step %}
#### Выдели границы квартиры

1. Возьми в руку обычную деревянную палочку(stick).
2. Выдели первый блок квартиры кликнув по нему ПКМ
3. По диагонали выдели второй блок кликнув по нему ЛКМ

{% hint style="info" %}
Выделенный куб не должен выходить за пределы региона(используй клавиши F3 + G, чтобы увидеть границы) и не должен пересекаться с другими квартирами.
{% endhint %}
{% endstep %}

{% step %}
#### Выполни команду

`/flat create <цена>` - создать квартиру с определённой ценой
{% endstep %}

{% step %}
#### Выдай квартиру игроку

Если ты сделал квартиру для продажи, то дальше можно просто ждать пока её кто-нибудь купит. Если же надо сразу выдать игроку квартиру то выполни команду:

**`/flat member add <ник>`** - заселить игрока в квартиру

{% hint style="info" %}
В квартиру можно заселить максимум 3-ёх игроков
{% endhint %}

{% hint style="warning" %}
Покупать квартиры могут не только жители твоего города, но и любой житель из других городов
{% endhint %}
{% endstep %}
{% endstepper %}

После успешного выполнения команды квартира будет создана и ты сможешь увидеть её границы:

<figure><img src="../.gitbook/assets/image (39).png" alt=""><figcaption><p>Выделение границ квартиры</p></figcaption></figure>

Если ты забыл размер квартиры, то отобразить границы всегда возможно командой:

* **`/flat show`** - показать границы квартиры

## <mark style="color:$primary;">Управление квартирой</mark>

Для максимального удобства управления всеми городскими и своими квартирами существует меню:

* **`/flat`** - меню управления своими и городскими квартирами

<figure><img src="../.gitbook/assets/image (58).png" alt=""><figcaption></figcaption></figure>

### Владелец квартиры

У владельца квартиры есть больше прав по управлению квартирой чем у обычного жильца:

* Выставление квартиры на продажу
* Установленые цены на квартиру
* Добавление жильцов
