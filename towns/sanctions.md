# 🚧 Санкции

## <mark style="color:$primary;">Команды</mark>

* **`/t sanction set <город>`** - ввести против города санкцию
* **`/t sanction cancel <санкция> <город>`** - отменить санкцию
* **`/t sanctions`** - меню санкций города

## <mark style="color:$primary;">Основы</mark>

После достижения городом **2 уровня**, он имеен возможность вводить санкции против других городов:

* **`/t sanction set <санкция> <город>`** - ввести против города санкцию

После ввода команды тебе будет предложено выбрать время действия санкции: минимум 1 день и максимум 30 дней. После окончания этого срока санкция будет удалена.

{% hint style="info" %}
Городу доступно одновременно только **10** активных санкций, поэтому стоит хорошо продумывать свои действия.
{% endhint %}

Санкцию можно отменить в любой момент:

* **`/t sanction cancel <санкция> <город>`** - отменить санкцию

## <mark style="color:$primary;">Типы санкций</mark>

| Тип              | Описание                                                                                                                                                               |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Эмбарго**      | Жители твоего и санкционного городов не смогут покупать друг у друга товары на мировом рынке и в честшопах. Товары у скупщика дорожают на +3% за каждую такую санкцию. |
| **Террабарго**   | Увеличивает стоимость привата регионов санкционного города на &#x35;**%**                                                                                              |
| **Констрабарго** | Увеличивает стоимость покупки зданий санкционного города на **3%**                                                                                                     |
