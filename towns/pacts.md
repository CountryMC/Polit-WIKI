# 🎗️ Пакты

## <mark style="color:$primary;">Команды</mark>

* **`/t pacts`** - меню пактов города
* **`/t pact set <город>`** - отправить городу запрос на заключение пакта
* **`/t pact cancel <type> <город>`** - расторгнуть пакт с городом

## <mark style="color:$primary;">Основы</mark>

С любым городом, достигшим **2 уровня** можно заключать пакты. Каждый тип пакта имеет свою уникальную особенность.

* **`/t pact set <город>`** - отправить городу запрос на заключение пакта

Каждый пакт заключается минимум на 1 день и максимум на 30 дней(выбирается игроком в меню), но его можно расторгнуть, заплатив штрафную сумму из казны города.

* **`/t pact cancel <пакт> <город>`** - расторгнуть пакт с городом

{% hint style="success" %}
Житель с рангом Дипломат может только заключать пакты
{% endhint %}

## <mark style="color:$primary;">Типы пактов</mark>

|     Название    |                                                              Особенность                                                             | Штраф за расторжение |
| :-------------: | :----------------------------------------------------------------------------------------------------------------------------------: | :------------------: |
|    **Защита**   | Оба города автоматически вступают на сторону защиты при рейде на одного из участников пакта независимо от наличия общего государства |        **300**       |
| **Ненападение** |                                        Оба города не смогу рейдить и воевать друг против друга                                       |        **600**       |
