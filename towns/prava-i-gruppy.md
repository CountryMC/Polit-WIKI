# 🕵️‍♂️ Права и группы

Права определяют, кто может выполнять определённые действия на территории города, участка, группы участков или в персональных настройках жителя.

## <mark style="color:$primary;">Области действия разрешений</mark>

Все разрешения работают одинаково, отличается только область их применения.

| Тип             | Команды                                     | Область действия             |
| --------------- | ------------------------------------------- | ---------------------------- |
| Город           | `/t perms`, `/t set perm`                   | Весь город                   |
| Участок         | `/plot perms`, `/plot set perm`             | Текущий участок              |
| Группа участков | `/plot group perms`, `/plot group set perm` | Все участки в группе         |
| Житель          | `/res perms`, `/res set perm`               | Настройки конкретного жителя |

## <mark style="color:$primary;">Группы игроков</mark>

Для каждого разрешения можно отдельно настроить доступ следующим группам:

| Группа          | Описание                              |
| --------------- | ------------------------------------- |
| **friend**      | Друг                                  |
| **town-friend** | Друзья города (/t friend add <город>) |
| **resident**    | Житель города                         |
| **ally**        | Союзник из вашего государства         |
| **outsider**    | Любой посторонний игрок               |

## <mark style="color:$primary;">Команды</mark>

Вариации использования комманд:

* **`<команда> on/off`** — включает/выключат все права для всех групп
* **`<команда> <группа> on/off`** — включает/выключат все права для указанной группы
* **`<команда> <действие> on/off`** — включает/выключат определённое действие для всех групп
* **`<команда> <группа> <действие> on/off`** — включает/выключат определённое действие для определённой группы

### Примеры

* `/t set perm build residents on` - <mark style="color:$success;">разрешить</mark> всем жителям своего города ставить блоки на городских участках(без владельцев)
* `/plot set perm destroy friend on` - <mark style="color:$success;">разрешить</mark> своим друзьям разрушать блоки на участке, в котором находишься
* `/plot group <группа> set perm switch resident on` - <mark style="color:$success;">разрешить</mark> своим друзьям разрушать блоки на участках, которые состоят в определённой группе
* `/res set perm item_use nation off` - <mark style="color:$danger;">запретить</mark> использование предметов жителям государства на участках, в которых являешься владельцем

## <mark style="color:$primary;">Меню настройки</mark>

<figure><img src="../.gitbook/assets/image (29).png" alt=""><figcaption><p>Меню городских праф /t perms</p></figcaption></figure>

Каждый тип разрешений имеет собственное меню настройки:

* **`/t perms`** — разрешения города
* **`/plot perms`** — разрешения участка
* **`/plot group perms`** — разрешения группы участков
* **`/res perms`** — персональные разрешения жителя

В меню можно изменять разрешения для каждой группы игроков без использования команд.

## <mark style="color:$primary;">Разрешения</mark>

| Разрешение    | Описание                                                                        |
| ------------- | ------------------------------------------------------------------------------- |
| **build**     | Размещение блоков                                                               |
| **destroy**   | Ломание блоков                                                                  |
| **switch**    | Использование переключаемых блоков(дверей, люков, кнопок, рычагов и тп)         |
| **item\_use** | Использование предметов(лодки, вагонетки, эндерпёрлы, хорусы, феерверки, лоссо) |
