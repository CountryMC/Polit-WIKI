# ☠️ Войны

## <mark style="color:$primary;">Команды</mark>

* **`/t war`** - меню твоей текущей войны
* **`/t neutrality`** - меню продления нейтралитета
* **`/t wars`** - меню с информацией о войнах на сервере
* **`/t war declare force <город>`** - объявить войну городу с просроченым нейтралитетом
* **`/t war declare offer <город>`** - предложить войну городу независимо от нейтралитета
* **`/t war army`** - меню армии
* **`/t war join`** - присоединиться к армии союзника
* **`/t war leave`** - покинуть армию
* **`/t war kick`** - выгнать игрока из армии
* **`/t war leave`** - покинуть армию
* **`/t war time`** - время до начала/конца войны
* **`/t war result`** - меню выбора результата войны
* **`/t war result <тип>`** - установить результат войны

## <mark style="color:$primary;">Типы войн</mark>

### <mark style="color:$info;">Принудительная война</mark>

Начиная с **3 уровня** каждому городу автоматически выдаётся **5 дней** нейтралитета, который необходимо продлевать когда он подходит к концу или когда полностью истёк. Если время нейтралитета было просрочено и не продлено, любой город достигший **4 уровня** и имеющий активированное здание [#citadel](buildings.md#citadel "mention") сможет объявить этому городу принудительную войну.

Следить за нейтралитетом можно в городском меню: **`/t`** или **`/t neutrality`**

Продлить нейтралитет можно командой:

* **`/t neutrality`** - меню продления нейтралитета

Объявить принудительную войну можно командой:

* **`/t war declare force <город>`** - объявить войну городу с просроченым нейтралитетом

{% hint style="warning" %}
Объявлять принудительную войну можно только в период **15:00 - 19:00 МСК**. Также объявить войну можно только если с момента создания твоего города прошло **5 дней**.
{% endhint %}

Подробнее о проведении войны: [#osnovy-voiny](wars.md#osnovy-voiny "mention")

### <mark style="color:$info;">Договорная война</mark>

Начиная с **3 уровня** города любой город может предложить войну любому другому городу, достигшему **2 уровень**. Война начинается, если мэр города, которому была предложена война, согласится на это предложение.

Объявить договорную войну можно командой:

**`/t war declare offer <город>`** - предложить войну городу

{% hint style="success" %}
Других ограничений, лимитов и требований на проведение договорных войн нет. Воевать можно хоть 10 раз в день.
{% endhint %}

Подробнее о проведении войны: [#osnovy-voiny](wars.md#osnovy-voiny "mention")

## <mark style="color:$primary;">Основы войны</mark>

Война состоит из трех этапов: <mark style="color:orange;">**Подготовка**</mark>**&#x20;->&#x20;**<mark style="color:orange;">**Сражение**</mark>**&#x20;->&#x20;**<mark style="color:orange;">**Итог**</mark>

Для победы в войне надо набрать **200 очков**, либо набрать очков больше чем у противника в момент окончания этапа <mark style="color:orange;">Сражение</mark>. Если очков поровну - побеждает сторона защиты.

Победить в войне может любая из сторон, которая в итоге будет решать, какой итог будет с проигравшим городом.

### <mark style="color:$info;">**Подготовка**</mark>

Этап подготовки начинается сразу после объявления войны и длиться 30 минут для договорных и 24 часа для принудительных войн

Во время этапа подготовки обе стороны должны набрать армию. Для этого житель города или государства должен присоединиться к армии:

* **`/t war join`** - присоединиться к армии союзника
* **`/t war leave`** - покинуть армию
* **`/t war kick`** - выгнать игрока из армии

По умолчанию, игрок может присоединиться к армии только после одобрения одного из жителей города с соответствующим правом [#rangi-i-prava](perms.md#rangi-i-prava "mention"). Чтобы игроки могли вступать в армию без разрешения, нужно переключить флаг города [#flagi](flags.md#flagi "mention")

Присоединиться или покинуть армию можно в любой фазе войны

{% hint style="danger" %}
Игроки, которые не состоят в армии, не смогут наносить урон и зарабатывать очки во время войны
{% endhint %}

{% hint style="success" %}
Игрок с одного аккаунта может состоять сразу в нескольких армиях и соответственно участвовать в нескольких войнах.
{% endhint %}

Так же это можно сделать через меню войны:

* **`/t war`**  - меню войны

<figure><img src="../.gitbook/assets/image (8) (2) (1).png" alt=""><figcaption><p>Меню войны</p></figcaption></figure>

{% hint style="warning" %}
Присоединиться к армии могут только жители города либо жители государства, у которых уровень города больше **первого**
{% endhint %}

{% hint style="danger" %}
Лимит армии у обеих сторон равен **40 игроков**
{% endhint %}

### <mark style="color:$info;">**Сражение**</mark>

Во время этого этапа стороны должны зарабатывать очки для победы в течении **2 часов**. Это можно сделать двумя способами: **захват регионов** и **убийство врагов**. Оба способа доступны как стороне атаки(захват регионов), так и защиты(освобождение своих регионов). Война заканчивается, когда одна из сторон набрала **200 очков**, либо по окончанию времени(победитель тот у кого больше очков).

{% hint style="warning" %}
Во время этого этапа мэры обоих городов не смогут пользоваться большинством команд по управлению городом.
{% endhint %}

{% hint style="danger" %}
Во время этого этапа в городе защиты невозможно будет ломать: спавнера, сундуки, шалкера, бочки и маяки.
{% endhint %}

#### <mark style="color:orange;">**Захват регионов**</mark>

Для захвата регионов понадобиться скрафтить военные флаги:

<figure><img src="../.gitbook/assets/Screenshot from 2026-01-18 21-33-39.png" alt=""><figcaption><p>Крафт флага Атаки</p></figcaption></figure>

<figure><img src="../.gitbook/assets/Screenshot from 2026-01-18 21-33-54.png" alt=""><figcaption><p>Крафт флага защиты</p></figcaption></figure>

Есть два типа военных флагов: флаг Атаки и флаг Защиты. Флаг Атаки используется для захвата регионов атакующей стороной, а флаг Защиты для освобождения захваченных регионов обороняющейся стороной.

Чтобы начать захват нужно поставить флаг в нужный регион, после чего даётся 25 секунд (180 секунд для центрального региона), чтобы защитить установленный флаг и захватить регион.

Чтобы помешать захвату флага, противник должен ударить флаг столько раз, сколько он будет иметь прочности, после чего сломать флаг.

{% hint style="info" %}
Обе стороны могут одновременно захватывать регионы, тоесть пока сторона атаки захватывает регион в одном месте, сторона защиты может освобождать другой регион в другом месте, но максимум по одному региону с каждой стороны, тоесть нельзя одновременно захватывать два и более регионов, так же как и освобождать два и более регионов.
{% endhint %}

{% hint style="warning" %}
Захватывать можно только регионы, рядом с которым граничит хотя бы один захваченный регион. В начале войны, когда ни один регион еще не захвачен, сторона атаки может начать захватывать только регионы, находящиеся по периметру защищающегося города.
{% endhint %}

{% hint style="warning" %}
Захватывать регионы могут только мэр и жители атакующего города с рангом военноначальник. Максимальное количество военноначальников равно **1**
{% endhint %}

**Очки за захват регионов**

* Захват обычного региона - **5 очка**
* Захват центрального региона атакующим городом - 35 **очков**
* Захват центрального региона защищающимся городом - 35 **очков**

#### <mark style="color:orange;">**Убийство врагов**</mark>

Очки за убийство начисляются только за убийство игрока состоящего в армии.

**Очки за убийство**

* Убийство обычного игрока - **1 очко**
* Убийство мэра атакующего/защищающегося города - **5 очков**
* Убийство военноначальника атакующего/защищающегося города - **3 очка**

### <mark style="color:$info;">**Исход**</mark>

Исход длится **30 минут** и начинается после достижения одним из городов максимального кол-ва очков или при истечении времени фазы Сражение. За это время мэр победившего города должен выбрать результат войны.

Всего есть **пять** возможных исходов войны: <mark style="color:orange;">**Уничтожение**</mark>, <mark style="color:orange;">**Покорение**</mark>, <mark style="color:orange;">**Разграбление**</mark>, <mark style="color:orange;">**Ограбление**</mark>, <mark style="color:orange;">**Пощада**</mark>.

|                        Исход                        | Описание                                                                                                                | Доступен       |
| :-------------------------------------------------: | ----------------------------------------------------------------------------------------------------------------------- | -------------- |
|  <mark style="color:orange;">**Уничтожение**</mark> | Защищающийся город уничтожается, а его баланс переходит на баланс атакующего города                                     | Атаке и Защите |
|   <mark style="color:orange;">**Покорение**</mark>  | Присоединить захваченные регионы к атакующему городу без учёта текущего лимита на регионы                               | Атаке          |
| <mark style="color:orange;">**Разграбление**</mark> | Удалить все захваченный регионы. Защищающемуся городу будет установлен кулдаун на приват регионов в течении **3 часов** | Атаке          |
|  <mark style="color:orange;">**Ограбление**</mark>  | Баланс Защищающегося города переходит на баланс атакующего города                                                       | Атаке и Защите |
|    <mark style="color:orange;">**Пощада**</mark>    | Оба города остаются в таком же положении как и до войны                                                                 | Атаке и Защите |

Чтобы выбрать исход войны, мэр победившего города должен выполнить команду:

* **`/t war result`** - меню выбора результата войны

<figure><img src="../.gitbook/assets/image (8) (2).png" alt=""><figcaption><p>Меню выбора результата войны</p></figcaption></figure>

После выбора исхода, он сразу же начинает действовать и война считается завершённой.

{% hint style="success" %}
После принудительной войны обоим городам устанавливается кулдаун 2 дня на объявление войн, который не проверяется, если позже будет объявлена договорная война. После договорной войны кулдауны не устанавливаются.
{% endhint %}

## <mark style="color:$primary;">Сравнение типов войн</mark>

| Особенность                                              | Принудительная                             | Договорная            |
| -------------------------------------------------------- | ------------------------------------------ | --------------------- |
| Доступное время объявления войны                         | **15:00-19:00 МСК**                        | **Любое время**       |
| Наличие игроков онлайн                                   | **Не нужно**                               | **Только мэр защиты** |
| Длительность этапа подготовки                            | **24 часа**                                | **30 минут**          |
| Необходимый уровень города атаки                         | **4**                                      | **2**                 |
| Необходимый уровень города защиты                        | **3**                                      | **2**                 |
| Необходимое здание                                       | [#citadel](buildings.md#citadel "mention") | **Не нужно**          |
| Необходимое вол-во дней с момента основания города атаки | **5 дней**                                 | **Не нужно**          |
| Кулдаун на объвление следующей войны                     | **2 дня**                                  | **Нет**               |
