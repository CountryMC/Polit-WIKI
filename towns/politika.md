# ♟️ Политика

## <mark style="color:$primary;">Городская вражда и дружба</mark>

Любой город может объявить своим другом или врагом любой другой город

#### Команды

* `/t politics` - меню политических действий города
* `/t friends` - меню друзей города
* `/t friend add <город>` - добавить город в друзья
* `/t friend remove <город>` - удалить город из друзей
* `/t enemies` - меню врагов города
* `/t enemy add <город>` - добавить город в список врагов
* `/t enemy remove <город>` - удалить город из списока врагов

{% hint style="info" %}
Для жителей дружественных городов можно отдельно настраивать [права](perms.md#gruppy-igrokov) на участках. Максимальное число городских друзей - 10
{% endhint %}

{% hint style="info" %}
Жители вражеских городов не смогут телепортироваться в ваш город, даже если включен городской флаг на [публичный телепорт](flags.md#spisok-flagov)
{% endhint %}
