# ✍️ Команды

<details>

<summary>Город</summary>

* **`/t`** - меню города
* **`/t deposit <число>`** - положить деньги в бюджет города со своего счета
* **`/t withdraw <число>`** - снять деньги со счета города
* **`/t delete`** - удалить город
* **`/t leave`** - покинуть город
* **`/t spawn`** - телепорт в город
* **`/t spawn <город>`** - телепорт в чужой город
* **`/t claim`** - заприватить регион
* **`/t unclaim`** - удалить регион
* **`/t invite <ник>`** - пригласить игрока в город
* **`/t join <город>`** - вступить в публичный город
* **`/t kick <ник>`** - выгнать игрока из города
* **`/t find`** - отправить запрос на поиск города
* **`/t new <город>`** - создание города, если хватает ресурсов, иначе показывает сколько ресурсов осталось
* **`/t tax`** - размер налога и время до его сбора
* **`/t set tax <число от 1 до 60>`** - установить размер налога в процентах
* **`/t set mayor <ник>`** - передать владедие городом другому игроку
* **`/t set spawn`** - перенести точку спавна города (/t spawn)
* **`/t set homeblock`** - перенести домашний регион города
* **`/t perms`** - меню управление правами города [perms.md](towns/perms.md "mention")
* **`/t set perm ...`** - управление правами города [perms.md](towns/perms.md "mention")
* **`/t flags`** - меню управления флагами города [flags.md](towns/flags.md "mention")
* **`/t toggle <флаг> on/off`** - включить/выключить флаг города [flags.md](towns/flags.md "mention")
* **`/t inventory`** - открыть городской инвентарь
* **`/t buildings`** - открыть меню городских зданий
* **`/t online`** - список жителей онлайн
* **`/t online <город>`** - список жителей онлайн для определенного города
* **`/t res`** - меню управления жителями
* **`/t info`** - информация о своём городе
* **`/t info <город>`** - информация об определённом городе
* **`/t upgrade info <уровень>`** - посмотреть сколько ресурсов нужно для определенного уровня
* **`/t upgrade`** - увеличить уровень города
* **`/t levels`** - окрыть меню уровней города
* **`/t friends`** - меню друзей города
* **`/t friend add <город>`** - добавить город в друзья
* **`/t friend remove <город>`** - удалить город из друзей
* **`/t enemies`** - меню врагов города
* **`/t enemy add <город>`** - добавить город в список врагов
* **`/t enemy remove <город>`** - удалить город из списока врагов
* **`/t tax time`** - оставшееся время до следующего сбора налога с городов
* **`/t home`** - телепорт на точку дома (/res set home)
* **`/t politics`** - меню политических действий города
* **`/t eco`** - меню экономики города
* **`/t pacts`** - меню пактов города
* **`/t pact set <город>`** - отправить городу запрос на заключение пакта
* **`/t pact cancel <type> <город>`** - расторгнуть пакт с городом
* **`/t raid start <город>`** - начать рейд на город
* **`/t raid join <сторона>`** - присоединить свой город к рейду
* **`/t raid time`** - информация о кулдаунах на рейды
* **`/t raid end`** - оставшееся время до конца рейда
* **`/t raid cancel`** - отменить рейд или контратаку(только на этапе подготовки)
* **`/t raids`** - меню текущих рейдов на сервере
* **`/t military`** - меню военной политики, где можно посмотреть кулдауны города на рейды
* **`/t flats`** - список квартир в городе
* **`/t sanction set <город>`** - ввести против города санкцию
* **`/t sanction cancel <санкция> <город>`** - отменить санкцию
* **`/t sanctions`** - меню санкций города
* **`/t politic system`** - меню выбора политического строя

</details>

<summary>Участки</summary>

* **`/plot claim`** - заприватить(купить) участок
* **`/plot unclaim`** - расприватить участок
* **`/plot give <ник>`** - выдать участок игроку без продажи
* **`/plot`** - меню участка на котором ты стоишь
* **`/plot flags`** - меню управления флагами участка [flags.md](towns/flags.md "mention")
* **`/plot toggle <флаг> on/off`** - включить/выключить флаг участка [flags.md](towns/flags.md "mention")
* **`/plot perms`** - меню управление правами города [perms.md](towns/perms.md "mention")
* **`/plot set perm ...`** - управление правами города [perms.md](towns/perms.md "mention")
* **`/plot set price <число>`** - установить цену продажи участка
* **`/plot set name <name>`** - установить название участка
* **`/plot forsale`** - выставить участок на  продажу
* **`/plot notforsale` -** снять участок с продажи
* **`/plot group ...`** - управление группами участков [#gruppy-uchastkov](towns/plots.md#gruppy-uchastkov "mention")

#### Подробнее про участки [plots.md](towns/plots.md "mention")

</details>

<summary>Государства</summary>

* **`/nation new <название>`** - создание союза
* **`/nation`** - открыть меню государства
* **`/nation delete`** - удаление государства
* **`/nation invite <город>`** - пригласить город в государство
* **`/nation kick <город>`** - удалить город из государства
* **`/nation leave`** - покинуть государство
* **`/nation set capital <город>`** - передать власть государством другому городу
* **`/nation set color <цвет>`** - установить цвет государства на карте. Цвет должен быть HEX формата, например /union color #123ABC
* **`/nation list`** - список городов государства
* **`/nation towns`** - открыть меню городов государства
* **`/nation rename`** - переименовать государство
* **`/nation capital`** - телепорт в столицу государства
* **`/nationon residents`** - список всех жителей государства
* **`/nation online`** - показывает жителей государства в сети
* **`/nation online <государство>`** - показывает жителей государства в сети для определенного государства

#### Подробнее про государства [options.md](nations/options.md "mention")

</details>

<summary>Квартиры</summary>

* **`/flat`** - меню управления квартирой в которой стоишь
* **`/flat create <название>`** - создать квартиру с определённым названием
* **`/flat delete`** - удалить квартиру
* `/flat forsale` - выставить/снять квартиру на продажу
* **`/flat set price <цена>`** - установить цену квартиры
* **`/flat set name <цена>`** - изменить название квартиры
* `/flat member add <ник>` - добавить жильца в квартиру
* `/flat member remove <ник>` - удалить жильца из квартиры
* `/flat owner set <ник>` - сделать игрока владельцем квартиры
* `/flat owner clear` - удалить владельца из квартиры, но не выселить
* **`/flat buy`** - купить квартиру
* **`/flat show`** - показать границы квартиры
* **`/flat move`** - передвинуть голограму в то место, в котором находишься

</details>

<details>

<summary>Чаты</summary>

* **`/ch t`** или **`/tc`** - перейти в городской чат
* **`/ch l`** или **`/lc`** - перейти в локальный чат(радиус **96** блоков)
* **`/ch n`**  или **`/nc`** - перейти в государственный чат
* **`/ch g`** или **`/g`** или **`!`** - перейти в глобальный чат
* **`/msg <ник> <сообщение>`** - отправить личное сообщение игроку

</details>

<summary>Полезное</summary>

### Порталы

* **`/owl next END/NETHER`** - посмотреть оставшееся время до открытия порталов
* **`/owl endtime END/NETHER`** - посмотреть оставшееся время до закрытия порталов
* **`/owl bonustime END/NETHER`** - посмотреть бонусное время

### Помощь

* **`/menu`** - меню сервера
* **`/time`** - посмотреть игровое время
* **`/stats`** - посмотреть свою статистику
* **`/help`** - меню помощи
* **`/rules`** - правила

### Другое

* **`/sm toggle`** - Включить/Выключить режим быстрой рубки дерева топором
* **`/fish menu`** - Меню рыбалки
